# ☕ Alaa Mohammed — Portfolio Website

Professional Barista & Branch Manager portfolio website with a luxurious coffee theme, dark mode support, and full content management system.

## 🌟 Features

### 🎨 Design & UI
- ☕ **Coffee-luxury warm color theme** — Brown, cream, and gold palette inspired by coffee culture
- 🌙 **Dark mode toggle** — Seamless light/dark theme switching with localStorage persistence
- 📱 **Mobile-first responsive design** — Optimized for all screen sizes (320px to 4K)
- 🎯 **Smooth scroll navigation** — Active section highlighting in navbar
- 🖼️ **Professional image gallery** — Lightbox-enabled photo showcase with hover effects
- 🔄 **Logo auto-switching** — Dark/light logo variants based on theme

### ⚡ Interactivity & Animations
- ✨ **Scroll reveal animations** — Elements animate on scroll with stagger effects
- 📊 **Animated stat counters** — Numbers animate from 0 to target value
- 🎯 **Skill progress bars** — Animated width transitions on scroll
- 💫 **Hover effects** — Cards, buttons, and images with smooth transitions

### 📄 Content Sections
- 👤 **Hero section** — Animated profile photo, stats, and call-to-action buttons
- 📖 **About section** — Bio, location, education, and languages
- 💼 **Experience timeline** — Work history with dates and descriptions
- ⭐ **Skills showcase** — Progress bars with icons and percentages
- 🏆 **Certificates** — Achievement cards with verification badges
- 🎓 **Education** — Academic background
- 🖼️ **Gallery** — Coffee artistry portfolio with lightbox
- 📧 **Contact section** — Multiple contact methods + working form

### 🔧 Technical Features
- 🔒 **Protected admin panel** — Login-protected content management
- 📊 **Dynamic content rendering** — All content loaded from data.js
- ⚡ **Lazy loading images** — Performance-optimized gallery
- ⌨️ **Keyboard navigation** — ESC to close lightbox
- 🎯 **SEO optimized** — Meta tags, Open Graph, Twitter Cards
- 🏷️ **Favicon support** — Multiple icon sizes + webmanifest

### 📱 Contact Integration
- 💬 **WhatsApp float button** — Always-visible chat button
- 📧 **Contact form** — Ready-to-use message form
- 🔗 **Social links** — Facebook, Instagram, WhatsApp integration
- 📄 **CV download** — Direct PDF download button


## 🛠️ Browser Support
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
